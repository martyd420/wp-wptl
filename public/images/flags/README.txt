Flag Images for WP Multilingual Translator
=============================================

This directory should contain flag images for each language you want to support.
The images should be named according to the country code (e.g., us.png, gb.png, de.png, fr.png, etc.).

Image Requirements:
- Format: PNG with transparency
- Size: 24x16 pixels (width x height)
- Naming: Use the ISO 3166-1 alpha-2 country code as the filename (lowercase)

Example files:
- us.png - United States
- gb.png - United Kingdom
- de.png - Germany
- fr.png - France
- es.png - Spain
- it.png - Italy
- pt.png - Portugal
- ru.png - Russia
- cn.png - China
- jp.png - Japan

You can find flag images online or create your own. Make sure you have the right to use the images.

The plugin will automatically use these images in the language switcher when the "Flags" style is selected.
